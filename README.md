# ONE桌遊開店專案管理系統

**完整的施工專案管理工具 | 適用於多店開業協調**

---

## 📖 線上使用

👉 **[立即開啟系統](https://ken0900101318-beep.github.io/one-project-management/)**

---

## 🎯 系統功能

### 1. 📊 儀表板
- 即時總覽：進行中案場、本月開業數、待處理/逾期任務
- 案場進度條：視覺化顯示每個專案完成度
- 本週關鍵任務：重要事項提醒

### 2. 🏢 案場管理
- 完整案場列表
- 當前階段追蹤
- 負責業務分配
- 狀態標籤（正常/延遲/逾期）

### 3. 📅 時程甘特圖
- 視覺化施工時程（D1-D45）
- 各階段時間軸展示
- 進行中/已完成/待進行 狀態標示
- 支援多案場同時顯示

### 4. 📦 物料追蹤
- 物料下單記錄
- 預計到貨日期
- 實際到貨日期
- 逾期自動預警

### 5. 👷 師傅排程
- 工種分類（隔間/配電/木工/壁紙/設備配對）
- 本週排程一覽
- 師傅聯絡方式
- 狀態管理（進行中/待確認/已確認）

### 6. ✅ 檢查清單
- 分階段標準檢查項目
- 即時勾選儲存
- 案場切換查看
- 支援匯出 PDF（功能預留）

## 🎨 系統特色

- ✅ **響應式設計**：手機、平板、電腦都能完美顯示
- ✅ **即時互動**：無需重新整理，資料即時更新
- ✅ **視覺化呈現**：進度條、甘特圖、色塊標籤

## 💻 使用方式

### 方式一：線上直接使用
1. 點擊上方連結開啟系統
2. 選擇功能模組（儀表板/案場管理/時程/物料/師傅/檢查清單）
3. 查看資料、勾選清單
4. 資料儲存在瀏覽器本地（LocalStorage）

### 方式二：加入手機主畫面
1. 用手機瀏覽器開啟系統
2. 點擊「分享」或「⋯」選單
3. 選擇「加入主畫面」
4. 像 APP 一樣直接開啟使用

## 📋 施工流程五大階段

### 階段一：簽約與規劃（D1-D7）
- 簽約、格局圖、水電圖
- 座位編號、包廂命名
- 報價與需求討論

### 階段二：設計與備料（D3-D14）
- 門牌燈製作（大陸）
- 電箱製作 + 門片
- APP 店家創建
- 倉儲物料準備
- 叫貨（平板燈、循環扇）

### 階段三：施工協調（D8-D30）
```
拆除（D8-D10）
  ↓
隔間單封（D11-D15）
  ↓
配電埋線（D16-D18）
  ↓
隔間雙封（D19-D23）
  ↓
木工 + 壁紙 + 配電收尾（D24-D28）
```

### 階段四：設備進場（D25-D35）
- 集運物品到貨（門牌燈、繳費機、紙杯）
- 麻將桌 + 軟裝進場
- 技師配對（門鎖、喇叭、網關）
- 繳費機組裝

### 階段五：驗收與開業（D35-D45）
- 內部驗收
- 業主驗收
- 系統測試
- 員工培訓
- 正式開業

## 👥 角色分工

### 業務（專案負責人）
- 全流程統籌協調
- 每日巡場回報進度
- 師傅排程確認
- 物料到貨追蹤

### 主管（專案監督）
- 進度監控、品質把關
- 週報審核、問題解決
- 關鍵節點介入

### 老闆（決策者）
- 掌握全局
- 每週統計報告
- 重大決策仲裁
